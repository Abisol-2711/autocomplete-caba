# Proyecto para testear Autocompleter-CABA

### Herramientas necesarias para levantar el proyecto

- VSCODE
- NPM
- NODEJS

### Pasos para levantar el proyecto

1. Clonar repo

```
git clone https://github.com/Abisol-2711/autocompleter-caba.git
```

2. Ir a la carpeta

```
cd autocompleter-caba
```

3. Instalar dependencias

```
npm i
```

4. Agregar las URLs a cada una de estas variables, las mismas estar√°n ubicadas en el archivo **.env** que deben crear

```
PROD = ''
CLIENT_ID_PROD =  ''
CLIENT_SECRET_PROD = ''
 
QA = ''
CLIENT_ID_QA =  ''
CLIENT_SECRET_QA= ''
 
HML = ''
CLIENT_ID_HML = '' 
CLIENT_SECRET_HML = ''
 
DEV = ''
```

5. Levantar proyecto

```
npm start
```
